About Me

Hi! I’m Dilip Bist, a passionate Junior Front-End Developer with hands-on experience in crafting responsive, user-friendly, and visually appealing websites. Over the past few months, I’ve built more than 20 web projects, turning creative ideas and Figma designs into fully functional interfaces that focus on both performance and user experience.

I have a strong foundation in HTML5, CSS3, JavaScript, and jQuery, along with experience using Bootstrap and Tailwind CSS to build responsive layouts efficiently. I enjoy working with Figma to translate UI/UX designs into clean, maintainable code — ensuring that every element looks great and works seamlessly across devices and browsers.

Beyond coding, I’m deeply interested in understanding how design and functionality come together to create smooth, engaging web experiences. My workflow involves writing clean, organized, and reusable code, optimizing performance, and following modern web standards.

I’ve also explored WordPress development, customizing themes and building front-end interfaces using PHP and CSS, which has helped me understand how front-end and back-end systems connect. I’m always eager to learn new technologies, improve my design-to-code process, and grow into a more complete developer. My goal is to contribute to a creative team where I can continue building modern, impactful web solutions while improving my skills every day.

When I’m not coding, I enjoy exploring new UI trends, improving my portfolio, and experimenting with interactive website animations and effects.
